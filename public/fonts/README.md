# Fonts

The site is wired for Sarvam's exact typefaces. Place the licensed .woff2 files
here with these exact names:

Display (headlines / hero / section titles):
- SeasonMix-Regular.woff2
- SeasonMix-Medium.woff2
- SeasonMix-Italic.woff2

UI & body (navigation, body, buttons):
- Matter-Regular.woff2
- Matter-Medium.woff2
- Matter-SemiBold.woff2

Until these files are present, the site falls back to Source Serif 4 and Inter
(loaded from Google Fonts) automatically. No code changes needed either way.
